
Prerequisites
=============

* Ruby >= 1.9.2
* Ruby gem `rake` (sudo gem install rake)
* Ruby gem `rainbow` (sudo gem install rainbow)
* LLVM 3.3 (install from source, see below)
* Clang >= 3.2
* Graphviz to render flow diagrams


Install
=======

* Build and install LLVM (see below)
* check `opt --version` -- it should be 3.3
* Adapt autoconf/configure.ac to suit you LLVM installation
* run autoconf/AutoRegen.sh
* run ./configure
* run make
* Run tests: rake test


Install LLVM
------------

* Clone git mirror: https://github.com/llvm-mirror/llvm
* git checkout release_33
* ./configure
* sudo make install ENABLE_OPTIMIZED=1 (builds and installs)
